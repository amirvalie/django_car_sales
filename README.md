# DjangoCarSales

## project setup


1- SetUp venv
```
virtualenv -p python3.10 venv
source venv/bin/activate
```

2- install Dependencies
```
pip install -r requirements_dev.txt
pip install -r requirements.txt
```

3- create your env
```
cp .env.example .env
```

4- Create tables
```
python manage.py migrate
```

5- run the project
```
python manage.py runserver
